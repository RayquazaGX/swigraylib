# swigraylib #

[SWIG](http://www.swig.org/) binding for [raylib](https://www.raylib.com/index.html)

## Current Status ##

- SWIG 4.0.2
- raylib 3.7
- Supported languages:
    - [x] Lua

## Pull Requests are welcomed ##

Would be nicer if this repo can be a collection featuring Ruby/Python/... bindings as well!
